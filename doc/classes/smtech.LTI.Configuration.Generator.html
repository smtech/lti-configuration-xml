\smtech\LTI\Configuration

\smtech\LTI\ConfigurationGenerator

Generate a valid LTI Tool Provider configuration XML document to facilitate tool placements within a Tool Consumer.

With minimal information (a name, ID and launch URL), the default configuration will provide a course navigation placement that uses the name as the link text and the launch URL as the the link URL, defaulting to 'anonymous' launch privacy.
